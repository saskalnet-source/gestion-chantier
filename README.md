# Application Listing Chantiers

Application simple en HTML pour gérer les chantiers/listings : recherche, ajout, modification, suppression, import CSV/Excel et export CSV.

## Mise en ligne avec GitHub Pages

1. Créez un dépôt GitHub, par exemple `listing-chantiers`.
2. Envoyez les fichiers `index.html` et `README.md` dans le dépôt.
3. Allez dans **Settings** > **Pages**.
4. Dans **Build and deployment**, choisissez :
   - Source : **Deploy from a branch**
   - Branch : **main**
   - Folder : **/root**
5. Cliquez sur **Save**.
6. GitHub donnera un lien du type : `https://votre-nom.github.io/listing-chantiers/`.

## Utilisation

- Les données ajoutées sont sauvegardées dans le navigateur de l'appareil.
- Pour partager les mêmes données entre plusieurs téléphones, il faudra ensuite ajouter une base de données en ligne.
